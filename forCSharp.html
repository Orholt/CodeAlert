Używanie FOR
Instrukcja FOR pozwala na sprawne sprawdzanie paru elementów na raz, tworzenie pętli logicznych oraz ma także wiele innych zastosowań. 
    // tak wygląda używanie instrukcji FOR:


    for ( < zmienna >; < warunek logiczny >; < postępowanie po pętli > )

Przykład użycia instrukcji for w celu liczenia do 100:


for (i=0;i< 100; i++ )
{
Console.Writeline(i.ToString());
}
