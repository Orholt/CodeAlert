Tworzenie funkcji
Funkcja pozwala enkapsulować kawałek kodu i wywoływać go z innych części kodu.\ Niedługo możesz napotkać sytuację, gdy będziesz musiał powtórzyć część kodu z wielu miejsc - to jest właśnie funkcja funkcji. W C# deklarujemy je właśnie w taki sposób: 
    // tak wygląda tworzenie funkcji:


    < widoczność > < typ zwrotu > < nazwa > (< parametry >)
    {
        < kod funkcji >
    }
Funkcję możemy wywołać w następujący sposób:
nazwafunkcji(parametry);

Dzięki takiemu wywołaniu skrypt wykona kod zawarty w danej funkcji, w tym przypadku wywoła funkcję o nazwie: "nazwafunkcji" wraz z parametrem (parametry), utwóżmy więc zatem takową funkcję.

//tworzymy funkcję nic nie zwracającą

public void nazwafunkcji(int parametry) {

//po otrzymaniu zmiennej "parametry"

Console.Writeline(parametry.ToString());

//pokazuje wartość tej zmiennej w konsoli po uprzedniej konwersji jej do wartosci string
}
