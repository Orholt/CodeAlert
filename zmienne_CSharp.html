Zmienne
Zmienne są najważniejsze w każdym języku programowania, to dzięki nim wszystko co zrobimy może się dynamicznie zmieniać.

Wyróżniamy następujące typy zmiennych:

    • Typy proste:

      • Część ze znakiem: sbyte, short, int``long

      • Całka bez znaku: byte, ushort, uint``ulong

      • Znaki Unicode: char

      • Binarny zmiennoprzecinkowy IEEE: float, double

      • Zmiennoprzecinkowa liczba dziesiętna o dużej precyzji: decimal

      • Wartość logiczna: bool

Sposoby przypisywania zmiennych:

< Typ zmiennej> < nazwa zmiennej>
Przykładowy kod z deklaracjami zmiennych:

function Main()
{
int x = 1;
string y = "1";
}

//teraz mamy doczynienie ze zmiennymi lokalnymi w funkcji Main.
//na potrzeby kursu zaadresujemy teraz zmienną globalną

public int x = 1;

//Dzięki tego typu adresacji mamy możliwość do odwołania się do zmiennej w obszarze całego kodu
