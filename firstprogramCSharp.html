Pierwszy program
Aby stworzuć pierwszy program w języku C# musimy najpierw zasygnalizować programowi, że będziemy korzystać z zasobów Systemowych.
Aby tego dokonać na samej górze naszego edytora kodu piszemy tak: 
    using System;

Następnie możemy przy pomocy funkcji Console.WriteLine() napisać coś w konsoli. 
 using System;

 Console.WriteLine("Hellow World!");

Można łatwo zauważyć, że taka konstrukcja sama w sobie nie będzie działać.
Naprawa tego zjawiska polega na deklaracji przestrzeni nazw, tak więc zróbmy. 
    using System;

    namespace program
    {

        public static void Main()
        {
            Console.WriteLine("Hellow World!");
        }

    }

Teraz jeśli otworzymy konsolę zobaczymy coś takiego:

      > Hellow World!
