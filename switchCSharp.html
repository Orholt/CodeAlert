Używanie instrukcji SWITCH
Instrukcja SWITCH pozwala na szybkie porównanie wartości w sytuacji kiedy pisanie wielu instrukcji IF byłoby zbyt uciążliwe.
 
    // tak wygląda używanie instrukcji SWITCH:


    switch ( < zmienna > )
    {
        case x:
        < akcja >;
        //w tym momencie kodu musimy koniecznie pamiętać o użyciu break;
        case y:
        < akcja >;
        break;
        case z:
        < akcja >;
        break;
        default: kiedy wartość jest różna od każdego z przypadków
        < akcja >;
        break;
    }

Przykład użycia instrukcji SWITCH w celu sprawdzenia czy x = 5:
    x = 5;

    switch (x)
    {
        case 1:
        Console.WriteLine("x=1");
        case 3:
        Console.WriteLine("x=3");
        case 5:
        Console.WriteLine("x=5");
    }
